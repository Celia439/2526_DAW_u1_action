\doxysubsection{Detailed Description}
Representa un producto dentro de un inventario, incluyendo sus datos básicos como nombre, unidades disponibles, descripción y precio. Permite realizar operaciones comunes como aplicar descuentos, calcular el costo total de una compra y gestionar el stock.
